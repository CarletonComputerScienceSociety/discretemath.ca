# Setup

## 1. Install Yarn

This [guide](https://classic.yarnpkg.com/en/docs/install/#mac-stable) will describe the best setup for you're operating system.

## 2. Install Dependencies

```bash
yarn install
```

## 3. Start the app

```bash
yarn run dev
```

# Commands

## Test

```bash
yarn run test
```

## Format

```bash
yarn run format
```

## Storybook

```bash
yarn storybook
```
